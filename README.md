# SteamWallpaperTournament

**SteamWallpaperTournament** — це десктопний застосунок на базі WPF, створений для об'єктивного оцінювання та ранжування "живих" шпалер зі Steam за принципом турнірної сітки.

## 🚀 Основні можливості
* **Турнірна система:** порівняння шпалер парами для вибору найкращої.
* **Попередній перегляд:** інтегрований плеєр для перегляду відеофайлів прямо в інтерфейсі.
* **Гнучке управління:** можливість швидкого вибору переможця та переходу до наступного етапу.
* **Фінальний рейтинг:** автоматичне формування списку лідерів за результатами всіх порівнянь.

## 🛠 Технологічний стек
* **Мова:** C#
* **Платформа:** .NET (WPF)
* **Медіа:** MediaElement для відтворення відео.
* **Архітектура:** MVVM (чисте розділення логіки та інтерфейсу).

## ⚙️ Як запустити

### Варіант 1: Для користувачів (готовий файл)
1. Перейдіть у розділ `Для профиля стим/bin/Debug/net10.0-windows` на сторінці репозиторію.
2. Завантажте усі файли з цієї папки
3. Запустіть файл `Для профиля стим.exe`.

### Варіант 2: Для розробників (збірка з вихідного коду)
1. Клонуйте репозиторій
2. Відкрийте файл рішення `.sln` у **Visual Studio 2022** або новішій версії.
3. Натисніть **F5** (Start) для збірки та запуску проєкту.
